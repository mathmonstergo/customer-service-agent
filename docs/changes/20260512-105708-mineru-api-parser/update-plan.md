# MinerU 第三方 API 文档解析接入计划

## 修改目标

把 MinerU 作为文档解析服务接入本项目导入审核流程。第一期固定使用 MinerU 官方云端批量精准 API，上传 PDF、Word、Excel 等文件后生成结构化文本切块，再沿用现有 AI 候选 FAQ 和人工审核流程。

## 影响范围

- 配置：新增 MinerU Token、超时和再处理开关；官方接口 URL 由代码层维护。
- 导入识别：PDF、Word、Excel 使用 `mineru` 解析器。
- 文档解析：新增 MinerU API 客户端，支持官方批量精准 API。
- 再处理层：新增可选 `mineru-kb-packager` 风格处理，提升 MinerU 输出进入知识库前的结构质量。
- 管理后台 API：上传只保存原件；文档管理显式提交 MinerU 解析任务并轮询动态状态；Markdown 微信聊天记录继续走现有解析器。
- 知识库 UI：新增轻量知识库主页，作为 FAQ 管理、文档管理和智能问答的入口汇总。
- 文档管理 UI：独立承载文档导入、原件管理、MinerU 解析状态和切片查看。
- FAQ 生成 UI：原导入审核页后续收进 FAQ 管理，作为“从已解析切片生成候选 FAQ”的子功能。
- 设置中心 UI：从本地运行配置读取解析器、LLM、Embedding 等字段；密钥默认掩码，可通过眼睛图标查看；模型名支持手动输入。
- 测试：补充 MinerU 解析适配层、配置和导入流程单元测试。

## 具体步骤

1. 新增 `customer_service_agent/document_parser.py`，封装 MinerU API 调用和输出归一化。
2. 新增配置项 `MINERU_API_TOKEN`、`MINERU_PARSE_TIMEOUT_SECONDS`、`MINERU_USE_KB_PACKAGER`，MinerU 官方批量接口 URL 固定在代码层维护。
3. 调整 `detect_file_type()`，让 PDF、docx、xlsx/xls 使用 `mineru` 解析器。
4. 调整导入上传和重新解析逻辑：`markdown_chat` 使用原流程，`mineru` 使用 MinerU 流程。
5. 为 MinerU 批量精准 API、zip 结果读取、再处理开关和导入流程写测试。
6. 更新参考文档，记录 MinerU Token 配置和代码层固定接口方向。
7. 调整解析块点击逻辑，避免点击切片直接跳到候选 FAQ 审核页。
8. 将知识库顶层改为轻量入口页：第一次点击入口卡片展开悬浮待处理面板，第二次点击同一卡片进入对应页面。
9. 新增正式文档管理页：上传只保存原件，文件行点击后打开顶层侧拉抽屉展示详情操作和切片预览。
10. 设置中心新增 `/api/settings` 快照读取，用于填充当前 env 配置；Chat Model 改为可输入字段并保留候选模型。
11. 拆分 MinerU 解析接口为“提交任务 / 查询状态 / 下载结果”，前端按 `state` 和 `extract_progress` 展示动态进度。
12. 文档管理页按用户参考图调整为文件表格 + 右侧抽屉；无异常切片时不显示失败切片区域。

## 预期效果

- 上传 PDF、Word、Excel 后，系统自动调用 MinerU 解析。
- MinerU 返回内容后，系统生成带来源信息的 `import_chunks`。
- 解析结果默认进入 `needs_review`，不会直接进入正式 FAQ 或向量检索。
- MinerU 云端解析固定走官方批量精准 API；用户只需要在设置页填写 Token。
- 用户可先阅读切片真实内容，再按需生成候选 FAQ；不是每个文档切片都默认需要沉淀成 FAQ。
- 知识库主页保持轻量，只展示搜索框、FAQ 管理、文档管理、智能问答三个入口和悬浮待处理摘要。
- 文档管理页上传文档后默认保持“未解析”，用户在侧拉详情中点击“开始解析 / 重新解析”才调用解析接口。
- MinerU 查询状态为动态轮询：解析中显示页数进度，完成后自动下载结果、生成切片并刷新切片查看。
- MinerU 通常返回文件级失败状态和错误信息；本项目仅在本地切片记录明确有失败状态时展示异常切片区域。
- 设置弹窗打开时填充当前运行配置，密钥显隐按钮展示真实字段值而不是静态星号占位。
- 设置弹窗内保存、测试或错误提示会浮在玻璃遮罩上方；密钥隐藏态统一显示固定 16 个黑点，不暴露真实 Key 长度。
- 设置保存已接入本地 `data/settings.local.json` 租户配置写回，只做必填和格式校验，不做外部 API 连通性测试；各分组测试按钮保留为独立测试入口。
- 未保存提示改为表单当前值与加载基线对比，选项切回原值后会自动消失。
- MinerU 接口 URL 不再由用户填写；批量上传和批量结果查询 URL 固定在代码层，后续官方变更时再更新代码。

## 用户确认

用户已确认：当前先不本地安装 MinerU，使用 MinerU 第三方服务；设置页里 MinerU 只填写 Token，接口 URL 固定在代码层维护。
